  1. CSS stands for Cascading Style Sheets.
  2. CSS describe how HTML elements are to be displayed on screen,paper.or in other media.
  3. Css is not necessary. CSS saves a lot of work.It can cantrol the layout of multiple web page all at once

CSS can be added to HTML documents in 3 ways :

  • Inline-by using the style attribute inside HTML elements
  • Internal - by using a style element in thr head section.
  • External - by using a link element to link an external CSS file.
  • The most comman way to add CSS, is to keep the styles in external CSS files.However, in this tutorial we will use inline and internal styles, because this is easier to demonstrare, and easier for you to try it yourself.
